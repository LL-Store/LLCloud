# [LLCloud](https://github.com/LL-Store/LLCloud)
🔥 文件或文件夹的上传、下载、ZIP文件压缩与解压、在线查看等

## 如何开发

开发的时候，直接运行如下命令：

```
npm run start:dev
```

然后浏览器访问： ```http://127.0.0.1:30000/index.html```

## 生产环境

开发完毕以后，需要发布的话，首先进行打包：

```
npm run build
```

然后启动：

```
npm run start:prod
```
